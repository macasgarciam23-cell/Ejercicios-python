# Ejercicios de Bucles en Python (Data Science)

Este proyecto contiene una serie de ejercicios básicos en Python enfocados en el uso de bucles `for`, condicionales y operaciones matemáticas.

## 📌 Archivo principal
- `bucle_for_basico1.py`

## 📚 Ejercicios desarrollados

1. **Básico**
   - Imprime todos los números enteros del 0 al 100.

2. **Múltiples de 2**
   - Imprime todos los números múltiplos de 2 entre 2 y 500.

3. **Contando Vanilla Ice**
   - Imprime números del 1 al 100.
   - Si el número es divisible por 5 → imprime "ice ice".
   - Si es divisible por 10 → imprime "baby".

4. **Número gigante**
   - Suma todos los números pares desde 0 hasta 500,000.
   - Muestra la suma total.

5. **Cuenta regresiva**
   - Imprime números desde 2024 hasta 0, restando de 3 en 3.

6. **Contador dinámico**
   - Imprime números múltiplos de un valor dentro de un rango definido.

## 🛠 Tecnologías utilizadas
